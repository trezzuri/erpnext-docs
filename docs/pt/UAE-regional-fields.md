# Campos regionais dos Emirados Árabes Unidos



**Campos adicionados aos DocTypes padrão para ajudar na conformidade regional.**


## 1. Mestre de Itens


1. **Tem classificação zero**: marque esta caixa de seleção para itens com classificação zero.
2. **Está isento**: marque esta caixa de seleção para itens isentos de impostos.


## 2. Fatura de vendas


1. **Emirado de IVA**: selecione o Emirado do local de fornecimento.
2. **Reembolso fornecido aos turistas**: insira o valor do imposto que foi reembolsado aos turistas.


## 3. Fatura de compra


1. **Despesa Recuperável com Classificação Padrão**: insira o valor do imposto que pode ser recuperado.
2. **Cobrança reversa aplicável**: Escolha 'Y' se a cobrança reversa for aplicável. Os impostos seriam zerados e um lançamento contábil reverso seria criado.
3. **Cobrança reversa recuperável (porcentagem)**: insira a porcentagem pela qual o imposto pago sob cobrança reversa é recuperável.
