# Acompanhamento de inventário serializado



Caso de uso:

  


Em um cenário de manufatura, o item é fabricado e os números de série são gerados assim que a entrada de estoque é enviada.

Mas se houver necessidade de criar a lista de números de série sem criação automática (dinamicamente conforme a necessidade). Como os números de série podem ser gerados?

  


Vá para Lista de itens--> Criar novo item--> Verifique **Tem número de série**  

  


![](/files/cPN7i2K.png)

  


 Vá para **Número de série** doctype--> Criar números de série

  


![](/files/3n83zxa.png) 

  


Durante a fabricação, Ordem de serviço--> Concluir--> Entrada de estoque--> Selecione os números de série de acordo com a quantidade de fabricação.

  


Isso permite selecionar números de série dinamicamente no momento da fabricação.
